转义字符

  • 定义：字符串中一些不太好表示的符号，比如换行、制表符等，联通一些需要特殊表示的符号，比如单引号，为了表示这些符号，我们需要一些特殊的形式，俗称转义，需要转义的字符，俗称转义字符。
  • 常用的转义字符:
    • 单引号： '
    • 双引号："
    • 响铃：\a
    • 反斜杠：\
    • 换行：\n
    • 水平制表符：\t
    • 倒退：\b
    • 空字符串。字符值为0：\0
    • unicode 16位的十六进制数值：\uxxxx (xxxx表示四个数字)
    • unicode 32位的十六进制数值：\uxxxx xxxx (xxxx xxxxx表示8位数字)
    • 十六进制值：\xXX (第一个小写的x是必须要有的，后面的大写X表示十六进制值)
    • 八进制值：\oXX (第一个小写的o是必须要有的，后面的大写X表示8进制值)

string类型的API

  • api定义：应用程序接口，是已经做好的第三方程序功能，我们只需要引用、调用即可，提高开发效率，减少开发时间。
  • API分类
    • 一般函数
      • startswith/endswith 判断以某特定字符开头或结尾
      • count 代表计数，计算一个字符从某一个字符串出现的次数。
      • find 查找字符串中要查找字符出现的位置，如果能查到，只返回字符出现的第一个位置。如果查不到，返回-1
  • 查询手册
    • 思路：遇到不熟悉的api时的思路：通过名字大致了解其功能，通过该api手册中的介绍了解是否满足需求，如果满足需求，进一步看手册寻找如何使用。如果没有现成的api满足需求后，需要自己写一个api
    • help(str) 查询字符串的api手册
    • 输入变量. 使用tab键显示出可以使用哪些函数,比如 a.

字符串的格式化

  • 把字符串按照我们想要的格式进行打印或填充
  • 格式化的方式分类
    • 使用%
      • 类似正则
      • %s 代表字符串
      • %r 字符串，表示正则，不是普通的字符串
      • %c 代表整数转换为单个字符
      • %d 代表数字（整数），如果输入是小数，只取整数
      • %u 代表无符号整数
      • %o 代表八进制
      • %x 代表十六进制 （小写x的时候，十六进制内容中字母要为小写）
      • %X 代表十六进制 （大写X，六进制内容中字母要为大写）
      • %e 代表浮点数（科学计数法，输入值必须要有e，且e为小写）例如3.2e+4
      • %E 代表浮点数（科学计数法，输入值必须要有E，且E为大写）例如3.2E+4
      • %f 代表浮点数（小数）,%与f中添加.1，代表小数点后保留一位数，%.2f 代表小数点后保留两位数
      • %g/%G 代表十进制形式的浮点或指数浮点的自动转换
      • 当格式化的数据与%后的输入值个数不一致时，会出现报错。因此格式化数据与输入值要一一对应。
    • 使用format函数
my name is zhangfulin

my name is zhangfulin ,my age is 28 , my height is 180.34cm
  • 字符串占位方法
    • %s或%d等格式化数据的输入值需要占位的话，可以在%与字母中间加上数字，例如：%5d 表明占5位，默认右对齐
    • 想要左对齐的话，就加上-。即%5d改为%-5d
    • 想要补0的话，将%5d改为%05d即可。
    • 对于浮点数需要占位的话，使用%f，并在%与f中添加.1，代表小数点后保留一位数，%.2f 代表小数点后保留两位数，从而满足占位的要求

      • 用地板除将两数相除，等到的结果取整
      • 如果是两个整数相除，得到的结果没有小数点
      • 如果是两个浮点数相除，得到的结果虽然也是取整，但是有小数点。
    • % 表示取余，也叫做取模
    • ** 表示幂指数运算
  • 运算优先级
    • ** 也就是幂指数运算是第一优先级
    • 乘/除/地板除/取模 是第二优先级
    • 加减法 是第三优先级
    • 同一优先级中，按从左往右的优先顺序进行运算
    • 有括号，优先计算括号里的
    • 如果混合运算中，有浮点数的参与，结果一定为浮点型

赋值运算符

  • 为python变量提供赋值运算功能的符号
    • = 表示给一个变量赋予一个对象（也就是值），当没有这个变量时，会创建这个变量并赋予值给他，当有这个变量的时候，就将变量的对象进行修改。
      • 可支持变量的批量赋值，多个变量用逗号分隔开，多个值也用逗号分隔开
      • 可支持两个变量值的交换 a,b=b,a
      • 除=外，其他的赋值，都是赋值和相应运算的简写形式。
    • += 表明a+=b 为a=a+b
    • -= 表明a-=b 为a=a-b
    • /= 表明a/=b 为a=a/b
    • %= 表明a%=b 为a=a%b
    • **= 表明a**=b 为a=a**b
    • //= 表明a//=b 为a=a//b

分支语句

  • 程序有三种结构构成：
    • 顺序：程序从上到下依次执行，一行执行完毕后才会执行下一行
    • 分支：通过判断条件，不同的结果对应不通用的处理程序段
    • 循环：多次重复执行相同或相似的代码
